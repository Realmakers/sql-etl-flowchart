# Project Summary
This project is a web application designed to convert SQL queries into visual ETL (Extract, Transform, Load) flowcharts. It simplifies data engineering tasks by allowing users to input SQL statements, which the application then translates into graphical representations of the ETL process. The primary aim is to enhance data manipulation and visualization, making complex data workflows more accessible and manageable for users.

# Project Module Description
- **SQL Parsing**: Analyzes SQL queries to extract components such as CTEs, tables, fields, and joins. It now includes improved handling for UNION ALL queries and ensures that all data sources are accurately represented.
- **Flowchart Visualization**: Utilizes ReactFlow to create ETL flowcharts based on parsed SQL. Enhancements include better visualization of UNION ALL relationships and clearer representation of subquery handling.
- **User Interface**: Features a SQL editor and flowchart display, providing a seamless experience for users to visualize their data workflows.
